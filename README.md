# Iletisim Kitabi

Redux Toolkit ile hazirlanmis basit bir iletisim kitabi. Kisi ekleme, filtreleme ve silme ozellikleri bulunur.

## Komutlar

- `npm run dev` - Gelistirme sunucusunu baslatir.
- `npm run build` - Uretim derlemesi alir.
- `npm run preview` - Derleme ciktilarini onizler.
- `npm run lint` - ESLint calistirir.
